# Tag

- 많이 쓰이는 태그들이 나와 있는 사이트
https://www.advancedwebranking.com/html/#tag

```
<h1></h1> 헤드 태그 : 제목(head)
<br> : 줄바꿈(line break)
<p></p> : 문단(Paragraph)
<li></li> : 목차(list)
```

## 부모태그, 자식태그
```
<ul> Unorderd list
    <li>
    </li>
</ul>

<ol> Ordered list
    <li>
    </li>
</ol>
```

## 문서 구조
```
<!doctype html> #관용적 표현
<html> #관용적 표현
<head>
</head> <!-- <body>를 설명함 -->

<body> <!-- 본문 -->
</body> 
</html>
```
